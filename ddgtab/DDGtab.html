Parameters of Non-Trivial Divisible Design Graphs

Below tables with parameters for non-trivial divisible design graphs generated on the basis of the restrictions.

The columns are:

  • existence
  • v - number of vertices
  • k - valency
  • λ1 - number of common neighbours of two vertices from the same class
  • λ2 - number of common neighbours of two vertices from different classes
  • m - number of classes in the canonical partition
  • n - size of classes in the canonical partition
  • θ1f1 - positive eigenvalue whose square is k-λ1, with multiplicity
  • θ2f2 - negative eigenvalue whose square is k-λ1, with multiplicity
  • τ1g1 - non-negative eigenvalue whose square is k2-λ2v, with multiplicity
  • τ2g2 - non-positive eigenvalue whose square is k2-λ2v, with multiplicity
  • comments
